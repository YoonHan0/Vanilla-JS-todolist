# Vanilla-JS-todolist
### 자바스크립트만을 이용한 To Do List 만들기📝

<br>

#### <현재까지 완성된 결과>
<img src="https://user-images.githubusercontent.com/87405950/204444032-c8110275-bdcb-4660-b986-8a1087381793.gif"/>

#### 구현된 기능
- Create : 리스트에 항목 추가
- Read : 리스트 불러와서 출력
- Delete : 'X' 클릭 시 해당 리스트 삭제
- Check List : 리스트 클릭 시 '✓' 체크 표시 활성화
